# Cricket Performance Dashboard using Power BI

## Overview

This project presents an interactive Power BI dashboard for analyzing cricket performance data. The dashboard provides insights into batting statistics, venue-wise performance, and match metrics using dynamic visualizations.

## Features

- KPI Cards
  - Total Matches
  - Total Innings
  - Total Runs
  - Batting Average
  - Centuries
  - Half Centuries

- Interactive Filters
  - Home/Away Venue

- Visualizations
  - Runs by Venue
  - Average by Venue
  - Matches vs Runs
  - 4s by Venue
  - 6s by Venue
  - 100s and 50s Comparison
  - Strike Rate by Venue
